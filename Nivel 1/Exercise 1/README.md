📄Descripció

Crea una classe anomenada NoGenericMethods que emmagatzemi tres arguments del mateix tipus, juntament amb els mètodes per a emmagatzemar i extreure aquests objectes i un constructor per a inicialitzar els tres.
Comprova que els arguments es poden posar en qualsevol posició en la crida al constructor.

💻Tecnologies Utilitzades

Se han utilizado Java Standard Library (JDK) y IntelliJ
